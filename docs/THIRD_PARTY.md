# Third-party code

This repo vendors upstream dependencies under `third_party/`:

- `third_party/mvsplat/`: MVSplat (Gaussian splatting NVS)
- `third_party/ELiC-ReImplemetation/`: ELIC reimplementation (learned compression)
- `third_party/diff-gaussian-rasterization-modified/`: CUDA rasterizer dependency

Each third-party component has its own license/notice files.
